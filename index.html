Rango

My first Django Project. This project is an implementation of the Rango app in TangoWithDjango book. Rango is a webpage directory which stores links to pages collected in a number of categories. Users can Like categories and see the pages in those categories in the order of views.

Users can also add pages and categories to Rango. The top 5 most viewed pages and most popular categories can be seen on the homepage.

This project was built to learn basics of Django. The implemented features include:

  • Implementation of User Registration, Login and Authentication.
  • Implementation of Bing Search API to search for pages and quickly add to the categories.
  • Implementation of AJAX to provide "Like" functionality, Suggestion based search and Quick Page adds.
